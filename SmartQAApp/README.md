# 智能问答APP

基于Kivy框架开发的智能问答应用程序，支持文字和语音交互。

## 主要功能

1. **多模态输入**
   - 文字输入
   - 语音输入（自动转为文本）

2. **智能理解与检索**
   - 词法分析
   - 句法分析
   - 语义理解
   - 多领域知识库检索

3. **智能回答生成**
   - 信息整合
   - 自然语言生成
   - 答案润色

4. **多模态输出**
   - 文本展示
   - 语音播报

## 界面功能

- 输入框：用户输入文字问题
- 语音输入按钮：支持语音提问
- 发送按钮：提交问题
- 历史记录按钮：查看历史问答记录
- 语音播报按钮：播放或关闭语音回答

## 安装与使用

1. 安装依赖：
```
pip install -r requirements.txt
```

2. 运行应用：
```
python main.py
```

## 目录结构

- `data/`: 存储知识库和配置文件
- `models/`: 机器学习模型和算法
- `utils/`: 工具函数和辅助模块
- `ui/`: 用户界面组件
- `resources/`: 资源文件（图标、音频等）
